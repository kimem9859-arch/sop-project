# 스마트 글라스 결선 초안

> 1인칭 카메라(글라스)의 **자체 전원·충전·안테나 결선**. 콘솔과는 **WiFi 무선**(물리 배선 없음).
> 설계 정본 = 통합문서 §5·§11. 작성 2026-06-12. (코드·배선도 = Claude / 배선·실연결 = 사용자)

## 1. 부품
| 부품 | 역할 |
|---|---|
| **XIAO ESP32-S3 Sense** | MCU + 카메라(OV3660) + WiFi |
| 450mAh LiPo (TW431575) | 전원 3.7V |
| **TP4056 충전모듈 SZH-LIP001** | 배터리 충전 (아래 선정 참조) |
| MSL-1C2P 스위치 | 전원 ON/OFF |
| 연호전자 2핀 1.25mm 케이블 | 배터리 커넥터(JST) |
| U.FL 안테나 | WiFi 거리 |

## 2. 충전모듈 선정 — SZH-LIP001 (확정)
BOM에 TP4056 2종 보유:
| 모듈 | 단자 | 보호출력 |
|---|---|---|
| **SZH-LIP001 (채택)** | +, −, B+, B−, **OUT+, OUT−** | ✅ 과방전·과전류 차단 |
| THC-CHGE01 | 5V+, GND, BAT+, BAT− | ❌ (BAT 직결, 보호 없음) |

→ **SZH-LIP001 채택**: OUT 단자가 **보호회로를 거친 출력**이라 LiPo 과방전 차단. (THC-CHGE01은 OUT 없어 비채택)

## 3. 핀-투-핀 결선 (SZH-LIP001)
```
[충전]   USB-C ─────────────→ SZH-LIP001 USB-C(IN)
[배터리] LiPo(+)빨강 ─(1.25mm)→ B+
         LiPo(−)검정 ─(1.25mm)→ B−
[출력]   OUT+ ─→ [MSL 스위치 공통] ─→ [ON측 단자] ─→ XIAO 5V핀
         OUT− ───────────────────────────────────→ XIAO GND핀
[안테나] U.FL 안테나 ─→ XIAO IPEX 커넥터
[카메라] Sense 카메라 = XIAO B2B 커넥터 장착(배선 X)
```

| SZH-LIP001 단자 | 연결 |
|---|---|
| +, − | 충전 입력(USB-C와 병렬) |
| B+, B− | 배터리 |
| OUT+, OUT− | 부하(XIAO) — 보호 출력 |

## 4. 전선 선정
글라스는 **웨어러블(머리 착용·움직임 많음)** → **연선 필수**(단선 금지: 굽힘 파단).

| 우선 | 전선 | 비고 |
|---|---|---|
| 1 (권장) | **실리콘 연선 26AWG** (UL3135/3239) | 유연·내열150℃+·납땜 쉬움 |
| 2 (가능) | PVC 연선 26AWG (UL2468 빨강/검정 쌍, UL1007 연선) | 사용 가능. 납땜 시 인두 오래 X(피복 수축) |
| ❌ 금지 | **단선(solid)** | 반복 굽힘에 구리심 파단 |

- 배터리측: 연호전자 **1.25mm 2핀 케이블**(보유)
- 연결: 작은 기기라 **납땜 + 열수축튜브** (WAGO·빵판 X)
- 전류 작음(~수백 mA) → 26AWG로 충분

## 5. 주의사항
1. **극성 엄수**: LiPo(+)→B+, (−)→B− (반대 시 손상)
2. **XIAO BAT 패드 미사용** — 충전은 TP4056가 전담(중복충전 방지)
3. **충전은 TP4056 USB-C로만** — XIAO USB-C는 프로그래밍/전원용
4. **스위치 ON측** = 정해진 게 아니라 공통+한쪽 선택 → 그쪽으로 밀었을 때 통전되는지 멀티미터로 확인
5. **모듈 OUT 패드 확인** — SZH-LIP001에 OUT+/OUT− 있음 확인(부하는 OUT에)

## 6. Pi 연결 = WiFi 무선
글라스는 카메라 영상을 **WiFi TCP로 Pi에 전송**(§5.2). 콘솔과 물리 배선 없음. WiFi 접속 흐름은 통합문서 §13.3.
