# 발표 테마 사양 — 기존 HTML 덱에서 추출

> 출처: `내용_준완성_애니메이션_사진확대프로젝트_상반기_프레젠테이션.html`(자체 제작 CSS, 외부 프레임워크 없음).
> 목적: **PPTX 마스터/테마를 이 디자인에 맞춰** 설정(스타일·테마만 가져오고 내용은 새로). 추출일 2026-06-13.
> ⚠️ HTML의 JS/CSS 애니메이션·사진확대는 PPTX로 자동 이전 안 됨 — PPTX 애니메이션으로 별도 적용. (이전: 색·폰트·레이아웃 / 미이전: 움직임)

## 1. 컬러 팔레트 (따뜻한 산업·안전 톤)
| 토큰 | HEX | 용도 |
|---|---|---|
| red | **#D63027** | 주 강조·위험·포인트(페이지번호 숫자, 배지) |
| red-dark | #B8241C | 레드 음영 |
| orange | **#E85D2F** | 보조 강조·영문 라벨·액센트 |
| orange-light | #F4A261 | 밝은 보조 |
| cream | **#FDF4ED** | **슬라이드 배경(기본)** |
| cream-deep | #FBE8D8 | 배경 장식 원·카드 보조 |
| ink | **#2B2B2B** | 본문·제목 텍스트 |
| gray | #6B6B6B | 보조 텍스트·페이지번호 |
| gray-light | #D8D2CC | 구분선·비활성 |
| white | #FFFFFF | 카드 배경·반전 텍스트 |

**그래디언트**: 액센트 바 = `orange→red`(180/90deg) · 헤더 액센트라인 = `orange→투명`(90deg).
**그림자**: 카드 = `0 2px 12px rgba(214,48,39,0.08)`(레드 틴트) · 슬라이드 = `0 20px 60px rgba(0,0,0,0.4)`.

## 2. 폰트
| 용도 | 폰트(우선순위) |
|---|---|
| 한글 본문·제목 | **Pretendard** → Apple SD Gothic Neo → Noto Sans KR |
| 영문·라벨·수치 | **JetBrains Mono** → SF Mono → Consolas (모노스페이스) |
| 라틴 본문 | Inter |
> ⚠️ Pretendard·JetBrains Mono·Inter는 **시스템에 설치 필요**(미설치 시 대체폰트로 렌더). 발표 PC에 설치 권장.

## 3. 타이포 스케일 (원본 vw → 16:9 PPTX pt 환산, 슬라이드 폭 960pt 기준 1vw≈9.6pt)
| 토큰 | vw | ≈ pt | 용도 |
|---|---|---|---|
| display | 3.5vw | ~34pt | 표지 대제목 |
| title | 2.5vw | ~24pt | 슬라이드 제목(h2, weight 800) |
| h3 | 1.5vw | ~14pt | 소제목·섹션번호 |
| h4 | 1.25vw | ~12pt | 카드 제목 |
| body | 1vw | ~10pt | 본문(글 최소 철학상 작게) |
| mini | 0.75vw | ~7pt | 영문 라벨·페이지번호 |
> 본문이 작게 느껴지면 1~2pt 상향 가능(시각 중심이라 텍스트는 보조).

## 4. 레이아웃 규격
- **화면비 16:9** (PPTX 와이드스크린 13.333"×7.5").
- **슬라이드 배경 = cream(#FDF4ED)**. 우상단·좌하단에 **cream-deep 원형 장식**(은은한 배경, z뒤).
- **슬라이드 헤더**: 좌측 [원형 섹션번호 배지] + [제목 h2(ink, 800) / 영문 부제(orange, mini, 자간 0.1em)] + 우측으로 뻗는 **액센트라인 3px(orange→투명)**.
- **본문 영역 패딩**: 상 0.5% / 좌우 5% / 하 4%.
- **페이지번호**: 우하단(bottom 2.2%, right 3%), gray·mini, **숫자만 red·bold**.
- **카드**: 흰 배경 + radius 8px + 레드틴트 그림자. 모서리 반경 4/8/12px.
- **공통 진입 애니메이션**(원본): 페이드 + 살짝 위로 슬라이드(translateY 20px→0). → PPTX에서 "밝기 변화/올라오기"로 대체.
