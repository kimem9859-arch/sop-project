# 슬라이드 마스터 빌드 사양서 (PowerPoint)

> **목적**: 이 문서를 PowerPoint 생성 도구에 그대로 주면 **요구대로 슬라이드 마스터·레이아웃**을 만들 수 있도록, 모든 값을 PowerPoint 좌표(인치·pt·HEX)로 명시한다.
> **출처**: 기존 HTML 덱 CSS에서 추출(테마 정본 = [`테마_사양.md`](테마_사양.md)). 내용은 무시, **스타일·테마·레이아웃만** 재현.
> **변환 기준**: 슬라이드 16:9 = **13.333" × 7.5"** (960pt × 540pt). 원본 CSS 1vw = 화면폭 1% = **0.1333" = 9.6pt** / 높이 1% = 0.075" = 5.4pt.
> ⚠️ HTML의 JS/CSS 애니메이션·사진확대는 PPTX로 이전 불가 → 필요 시 PowerPoint 애니메이션(밝기/올라오기)으로 별도 적용.

---

## A. 덱 설정

### A-1. 슬라이드 크기
- 16:9 와이드스크린, **폭 13.333in (33.867cm) × 높이 7.5in (19.05cm)**.

### A-2. 테마 색 팔레트 (PowerPoint 테마 색 슬롯에 매핑)
| PowerPoint 슬롯 | HEX | 원본 토큰 |
|---|---|---|
| Text/Background – Dark 1 (tx1) | **#2B2B2B** | ink (본문·제목) |
| Text/Background – Light 1 (bg1) | **#FFFFFF** | white (카드 배경) |
| Text/Background – Dark 2 (tx2) | **#6B6B6B** | gray (보조 텍스트) |
| Text/Background – Light 2 (bg2) | **#FDF4ED** | cream (슬라이드 배경) |
| Accent 1 | **#D63027** | red (주강조) |
| Accent 2 | **#E85D2F** | orange (보조강조) |
| Accent 3 | **#F4A261** | orange-light |
| Accent 4 | **#FBE8D8** | cream-deep (배경 장식) |
| Accent 5 | **#B8241C** | red-dark |
| Accent 6 | **#D8D2CC** | gray-light |
| Hyperlink | #E85D2F | orange |

### A-3. 테마 폰트
- **제목(Heading)·본문(Body) = Pretendard** (대체: Apple SD Gothic Neo, Noto Sans KR, 맑은 고딕).
- **수치·영문라벨·코드 = JetBrains Mono** (대체: Consolas) — 테마 슬롯 아님, 해당 요소에 수동 지정.
- ⚠️ 발표 PC에 Pretendard·JetBrains Mono **설치 필요**(미설치 시 대체폰트 렌더).

## B. 슬라이드 마스터(부모) — 모든 콘텐츠 레이아웃 공통

> 마스터 슬라이드 자체에 아래를 배치(모든 레이아웃이 상속).

1. **배경 채우기**: 단색 **cream #FDF4ED** (전면).
2. **배경 장식 원** (둘 다 cream-deep **#FBE8D8**, 윤곽선 없음, **맨 뒤 레이어**, 일부 화면 밖으로):
   - 우상단 원: 지름 **1.87in**, 위치 left **12.27in** / top **-0.80in**.
   - 좌하단 원: 지름 **1.33in**, 위치 left **-0.67in** / top **6.83in**.
3. **페이지 번호**(슬라이드 번호 placeholder): 우하단, **right 0.40in / bottom 0.165in** 기준 우측정렬 텍스트박스.
   - 글꼴 JetBrains Mono **7pt**, 색 gray #6B6B6B, 자간 +0.05em.
   - **현재 번호는 red #D63027 · Bold**, 나머지(예: "/ 24")는 gray. (표지·종료 레이아웃에는 미표시)

## C. 레이아웃 (총 6종)

### C-1. 표지 (Cover)
- 배경 cream. **페이지번호 없음.**
- 배경 장식 원(cream-deep): 우상단 지름 **2.93in**(top -0.6in/right -0.4in 부근), 좌하단 지름 **1.87in**(bottom 부근). 맨 뒤.
- 콘텐츠(좌측 정렬, 세로 가운데, 좌여백 0.67in):
  - **태그 배지**: orange #E85D2F 배경 · 흰 글자, padding 0.5vw×1.2vw, 글꼴 12pt 500 → 텍스트 예 `# Vision AI   # Edge AI   # Wearable`. (아래 24pt 여백)
  - **대제목 h1**: Pretendard **34pt · 800 · ink**, 줄간 1.25. (제목 텍스트)
  - **디바이더**: 폭 0.53in × 높이 3px(2.25pt), orange #E85D2F. (위아래 여백)
  - **부제 subtitle**: **14pt · orange · 500**.
  - **푸터**(우하단 bottom 0.225in/right 0.67in): 7pt gray — 소속·날짜 등.

### C-2. 콘텐츠 — 표준 (Content / 기본 본문) ★ 주력
**헤더 밴드** (상단, padding 위 3%=0.225in / 좌우 5%=0.67in / 아래 1.5%=0.11in, 요소 세로 가운데, 요소간 gap 1.2vw=0.16in):
  1. **섹션번호 배지 `num`**: **0.40in × 0.40in 원**, 배경 red #D63027, 흰 글자, **JetBrains Mono 14pt 800**, 가운데 정렬. (예 "01")
  2. **타이틀 그룹**(배지 우측):
     - **제목 h2**: Pretendard **24pt · 800 · ink**, 줄간 1.1.
     - **영문 부제 eng**: JetBrains Mono **7pt · orange #E85D2F · 500**, 자간 +0.1em, 제목 바로 아래(간격 0.2vw).
  3. **액센트 라인**: 제목그룹 우측 1vw(0.13in) 띄우고 **남은 폭 전부 채움**, 높이 3px(2.25pt), **그래디언트 orange→투명**(왼쪽 #E85D2F → 오른쪽 투명), 세로 가운데.

**바디 영역**(콘텐츠 placeholder): padding 위 0.5%=0.04in / 좌우 5%=0.67in / 아래 4%=0.30in.
  - 결과 사각형 ≈ **x 0.67in, y 0.95in, 폭 12.0in, 높이 6.25in**.

### C-3. 콘텐츠 — 2단/비교 (Two-Column)
- 헤더는 C-2와 동일.
- 바디를 좌우 2열로: 각 열 폭 ≈ **5.83in**, 열 간격 0.34in. (비교·대비, before/after용)

### C-4. 콘텐츠 — 큰 비주얼 (Big Visual)
- 헤더는 C-2와 동일.
- 바디 = **대형 이미지/다이어그램 placeholder**(바디 사각형의 ~80%) + 하단 캡션 1줄(10pt gray). (데모 캡처·시스템도·간트용)

### C-5. 섹션 구분 (Section Divider, 선택)
- 배경 cream + 장식 원.
- 중앙 좌측: 큰 섹션 라벨(예 "Ⅰ") **34pt 800 ink** + 섹션 제목 24pt + orange 디바이더(0.53in×3px).

### C-6. 종료 (Closing / Thank You)
- 배경 cream + 장식 원. 페이지번호 없음.
- 중앙: **"THANK YOU" 34pt 800 ink** (또는 그래디언트 orange→red 텍스트), 아래 부제 14pt orange. 하단 소속·조·날짜 7pt gray.

## E. 실행 지침 (PowerPoint 생성 도구에게)
1. A 설정(크기·테마 색 11슬롯·테마 폰트)을 먼저 적용.
2. B(마스터 부모) 공통 요소 배치 → 모든 레이아웃 상속.
3. C-1~C-6 레이아웃을 마스터 하위 레이아웃으로 생성(각 placeholder 위치·폰트·색 위 명세대로).
4. D 오브젝트는 빠른 재사용을 위해 프리셋/복제 가능한 도형으로.
5. **내용은 비우고 placeholder만** — 실제 텍스트·이미지는 [중간점검 대본](중간점검_발표_대본_1차.md)·[아웃라인](중간점검_발표_아웃라인.md) 기준으로 이후 채움.
6. 애니메이션은 미포함(필요 시 "올라오기+밝기" 진입 효과만 권장).

> 적용 후 점검: 색 11슬롯·폰트 2종·배경 cream·장식 원·헤더(번호배지+제목+영문부제+액센트라인)·페이지번호(숫자 red-bold)가 마스터에 들어갔는지 확인.
